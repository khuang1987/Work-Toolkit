# 桌面快捷方式创建说明

## 📌 如何将批处理文件添加到桌面

### 方法一：创建快捷方式（推荐）

1. **找到批处理文件**：
   - 打开文件夹：`python-scripts\line-tour-transfer\`
   - 找到文件：`运行巡线记录导入.bat`

2. **创建快捷方式**：
   - 右键点击 `运行巡线记录导入.bat`
   - 选择"发送到" → "桌面快捷方式"

3. **自定义图标（可选）**：
   - 右键点击桌面上的快捷方式
   - 选择"属性"
   - 点击"更改图标"
   - 选择一个您喜欢的图标

### 方法二：直接复制到桌面

1. 复制 `运行巡线记录导入.bat`
2. 粘贴到桌面

> ⚠️ **注意**：如果直接复制到桌面，路径可能会失效。建议使用方法一创建快捷方式。

## 🚀 使用方法

1. **双击桌面图标**即可运行
2. 程序会自动：
   - 读取上周工作日的巡线记录
   - 将数据写入到目标Excel文件
   - 打开Excel并定位到新添加的数据
3. 查看Excel确认数据无误
4. 按任意键关闭命令窗口

## 📋 批处理文件说明

文件位置：`python-scripts\line-tour-transfer\运行巡线记录导入.bat`

功能：
- ✅ 自动切换到正确的工作目录
- ✅ 使用虚拟环境中的 Python
- ✅ 运行数据导入脚本
- ✅ 完成后等待用户确认再关闭

## 🔧 故障排查

如果双击后报错：

1. **检查路径**：确保 bat 文件在正确的位置
2. **检查虚拟环境**：确保 `.venv` 文件夹存在
3. **查看日志**：打开 `transfer.log` 查看详细错误信息
