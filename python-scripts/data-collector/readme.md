# Toolkit DataCollector v1.4.1

**数据采集与处理工具**

一个专为数据收集和处理而设计的自动化工具，提供直观的界面和强大的功能，让数据管理工作变得简单高效。通过自动化处理文员每日重复的数据收集工作，将原本需要30分钟的手工操作缩短至几秒钟，避免人为错误，每年可节省120小时工作时间，相当于15个工作日，显著提升工作效率并释放人力资源。

## 🔧 技术架构

### 核心技术栈
| 组件 | 技术 | 说明 |
|------|------|------|
| GUI | Tkinter | Python 原生 GUI 框架 |
| 浏览器自动化 | Playwright | 现代化浏览器自动化工具 |
| 数据处理 | Pandas | 数据分析和处理 |
| 数据库 | SQLAlchemy + pyodbc | SQL Server 连接 |
| 系统托盘 | pystray | 系统托盘集成 |
| 进程管理 | psutil | 进程监控和管理 |

### 浏览器自动化说明
- 使用 **Playwright** 替代 Selenium WebDriver
- 复用 Edge 用户配置，无需重复登录
- 支持无头模式运行
- 自动处理文件下载

## ⚙️ 配置说明

### 路径配置
程序使用智能路径管理系统，支持以下路径类型：

**业务路径**
- `planner_folder` - Planner数据存储位置
- `transaction_folder` - 交易日志存储位置  
- `laborhour_folder` - 工时数据存储位置
- `product_folder` - 产量数据存储位置

**系统路径**
- `config_folder` - 配置文件目录
- `log_folder` - 日志文件目录
- `assets_folder` - 资源文件目录

### 配置方法
1. 启动程序后，点击"设置 > 路径配置"
2. 双击需要修改的路径进行编辑
3. 使用"检查路径"功能验证配置有效性

## 🎯 使用指南

## 🛠️ 故障排除

### 常见问题

**Q: 程序启动失败**
A: 检查是否安装了Microsoft Edge浏览器，确保有足够的系统权限。

**Q: 浏览器自动化错误**
A: 确保 Edge 浏览器已关闭所有窗口，程序需要独占使用用户配置。首次运行需执行 `playwright install msedge`。

**Q: 路径配置错误**
A: 使用"设置 > 路径配置"检查所有路径，确保文件夹存在且有访问权限。

**Q: PowerBI刷新失败**
A: 检查网络连接，确保PowerBI服务可访问，查看日志了解具体错误原因。
