📋 功能概述

数据采集工具是一款专为企业数据管理设计的自动化工具，基于 Python + Playwright 技术栈开发，支持从多个数据源自动采集、导出和刷新数据。

🌐 CMES 数据采集

自动从 Power BI 报表导出 CMES 生产数据，支持按月/年批量导出，智能检测缺失文件。

📅 Planner 任务导出

自动导出 Microsoft Planner 任务数据到 Excel，支持自动登录。

🔄 PowerBI 刷新

一键刷新本地 Power BI Desktop 报表数据，支持任务互斥保护。

⏰ 定时任务

设置定时任务，自动在指定时间执行数据采集，后台静默运行。

🔹 核心特性

  • 自动化采集 - 无需手动操作，自动完成登录、导航、数据导出
  • 智能登录 - 自动检测 Microsoft 登录页面，使用已保存账号登录
  • 断点续传 - 自动检测已存在文件，只导出缺失数据
  • 系统托盘 - 最小化到系统托盘，后台静默运行

🏗️ 技术架构

📐 技术栈

技术用途说明
Python 3.10+主开发语言跨平台、丰富的库支持
Playwright浏览器自动化支持 Chrome/Edge，比 Selenium 更稳定
Tkinter + ttkbootstrapGUI 界面原生 Windows 风格，轻量级
PyInstaller打包为 EXE单目录打包，便于分发
pystray系统托盘后台运行，桌面通知
psutil进程管理浏览器进程控制

💡 技术亮点

  • Playwright 持久化上下文 - 使用 Chrome 用户配置目录，保持登录状态
  • 任务锁机制 - 防止同一任务重复执行，支持中断后自动重置
  • 智能页面检测 - 多选择器检测页面加载，自动重试机制
  • 进程管理 - 只终止本程序启动的浏览器，不影响用户浏览器

🚀 快速开始

启动程序 - 双击 DataCollector.exe
系统自检 - 点击菜单 系统自检 检查环境
选择任务 - 勾选需要执行的任务
执行采集 - 点击 执行选中任务
💡 提示：首次使用请点击 设置 → 安装Playwright浏览器

⏰ 定时任务

支持设置每日定时任务，自动执行数据采集。

选项说明
启用定时刷新勾选后启用定时任务
执行时间设置每日执行时间（HH:MM）
任务选择选择要执行的任务类型
💡 提示：定时任务需要程序保持运行，可最小化到托盘。

❓ 常见问题

Q: 提示 Playwright 未安装？

A: 点击 设置 → 安装Playwright浏览器

Q: 数据采集失败？

A: 检查网络连接、Microsoft 账号登录状态、目标文件夹权限

Q: 如何中断任务？

A: 点击 中断任务 按钮，5秒后系统自动重置
