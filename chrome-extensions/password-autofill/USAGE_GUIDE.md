# 使用指南

## 快速上手

### 1. 打开插件
点击 Chrome 工具栏中的插件图标，打开弹出窗口。

### 2. 快速访问网页
直接点击系统名称（Windchill、Cornerstone、EHR）即可在新标签页中打开对应网页。

### 3. 配置设置
点击系统卡片右侧的 ▼ 图标，展开设置区域：
- **Windchill**: 配置用户名和密码，实现自动登录
- **Cornerstone**: 设置自动点击次数（1-100次）
- **EHR**: 配置用户名和密码，实现自动登录
- **定时启动**: 设置每天自动打开页面的时间

### 4. 保存设置
在设置区域填写完信息后，点击"💾 保存设置"按钮。

## 高级功能

### 右键菜单
在 Chrome 工具栏中右键点击插件图标，可以：
- 快速开启/关闭自动点击功能
- 快速开启/关闭自动关闭弹窗功能
- 🛑 中止所有插件指令（紧急停止）

### 键盘快捷键
- **Ctrl/Cmd + S**: 保存当前展开的设置
- **Enter**: 在输入框中按回车键快速保存

### 自动功能
1. **自动登录**: 
   - 配置账号密码后，访问对应系统时自动填充
   - 最多尝试3次，失败后弹窗提示

2. **自动点击** (Cornerstone):
   - 在右键菜单中开启
   - 自动完成课程进度
   - 可在设置中配置点击次数

3. **定时启动**:
   - 开启后每天在指定时间自动打开 Windchill 和 EHR
   - 默认时间为 09:00

## 常见问题

### Q: 如何停止正在运行的自动点击？
A: 右键点击插件图标，选择"🛑 中止所有插件指令"。

### Q: 登录失败怎么办？
A: 插件会自动尝试3次，失败后会弹窗询问是否继续尝试。请检查账号密码是否正确。

### Q: 如何在无痕模式下使用？
A: 进入 `chrome://extensions/`，找到本插件，点击"详细信息"，启用"在无痕模式下启用"选项。

### Q: 自动点击功能在哪里开启？
A: 右键点击插件图标，在菜单中切换"自动点击"开关。

## 安全提示

- 所有密码信息均使用 Chrome 安全存储机制保存
- 账号信息仅在本地保存，不会上传到任何服务器
- 插件仅在指定的 Medtronic 系统域名下运行
- 建议定期更新密码，保障账号安全

## 技术支持

如遇到问题，请检查：
1. Chrome 浏览器版本是否为最新
2. 插件是否已启用
3. 是否在支持的网站上使用
4. 浏览器控制台是否有错误信息
