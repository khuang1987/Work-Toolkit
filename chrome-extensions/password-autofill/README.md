# Medtronic Smart Assistant

Medtronic Smart Assistant 是一个Chrome浏览器扩展，旨在提高Medtronic员工日常工作效率。它提供了多个系统的快速访问、自动化填充及定时启动功能。

## 功能特点

### 1. 快速访问
- 一键访问Windchill系统
- 一键访问Cornerstone学习平台
- 一键访问EHR系统
- 美观的渐变色卡片式界面
- 点击系统名称直接打开网页

### 2. 自动填充功能
- **Windchill系统**: 
  - 支持网页表单和系统弹窗(Basic Auth)的自动登录
  - 智能重试：登录失败时最多尝试3次
  - 3次失败后显示 Toast 提示，不遮挡页面
  - 用户可选择继续尝试或取消
- **EHR系统**: 
  - 自动填充账号密码
  - 自动聚焦验证码输入框
  - 需要手动输入动态验证码
  - 不进行登录成功检测
- **安全存储**: 所有密码使用Chrome安全存储机制

### 3. Cornerstone自动化
- 自动点击功能（可选启用，通过右键菜单控制）
- 自动完成课程进度
- 可自定义点击次数（1-100次）
- 智能识别按钮状态

### 4. 每日定时启动
- 可开启"定时启动"功能
- 自定义启动时间（默认 09:00）
- 每天自动打开 Windchill 和 EHR 页面
- 开关式控制，方便启用/禁用

### 5. 紧急控制
- 右键菜单快速访问常用功能
- 一键中止所有正在运行的自动化操作
- 桌面通知确认操作完成

## 安装方法

1. 打开Chrome浏览器，进入扩展程序页面：
   - 在地址栏输入：`chrome://extensions/`
   - 或者点击菜单 -> 更多工具 -> 扩展程序

2. 启用"开发者模式"（右上角开关）

3. 安装扩展：
   - 点击"加载已解压的扩展程序"
   - 选择本插件所在文件夹

## 使用说明

### 快速访问和设置
1. 点击Chrome工具栏中的插件图标打开弹出窗口
2. 直接点击系统名称（如 Windchill、Cornerstone）即可快速打开对应网页
3. 点击右侧的 ▼ 图标可展开/收起设置区域
4. 在设置区域中配置账号密码等信息
5. 点击"💾 保存设置"按钮保存配置

### 键盘快捷键
- **Ctrl/Cmd + S**: 保存当前展开的设置
- **Enter**: 在输入框中按回车键快速保存

### 右键菜单功能
在Chrome工具栏中右键点击插件图标，可以快速访问以下功能：
- **自动点击开关**：快速启用/禁用Cornerstone自动点击功能
- **自动关闭弹窗开关**：快速启用/禁用弹窗自动关闭功能
- **🛑 中止所有插件指令**：立即停止所有正在运行的自动化操作
  - 停止自动点击
  - 停止登录重试
  - 清理所有运行状态
  - 适用于需要紧急中断插件操作的场景

### 弹出窗口功能
- **一键打开**：点击系统名称直接打开对应网页
- **展开设置**：点击右侧 ▼ 图标展开/收起设置区域
- **美观界面**：渐变色设计，卡片式布局，操作更直观
- **智能交互**：设置区域可展开收起，节省空间

### 系统设置
1. 点击"设置"标签页
2. 根据需要配置各系统：

#### Windchill设置
- 输入Windchill系统的用户名和密码
- 点击"💾 保存设置"按钮
- 访问 Windchill 时自动填充并登录
- 登录失败会自动重试，最多3次
- 支持处理系统原生登录弹窗

#### Cornerstone设置
- 设置自动点击次数（1-100次）
- 点击"💾 保存设置"按钮保存设置
- 启用/禁用自动点击功能请使用右键菜单

#### EHR设置
- 输入EHR系统的用户名和密码
- 点击"💾 保存设置"按钮
- 访问 EHR 时自动填充账号密码
- 自动聚焦验证码输入框
- 需要手动输入动态验证码

#### 每日定时
- 开启"每日自动打开页面"开关
- 插件将在每天 09:00 自动为你打开工作页面

## 安全说明

- 所有密码信息均使用Chrome安全存储机制保存
- 账号信息仅在本地保存，不会上传到任何服务器
- 插件仅在指定的Medtronic系统域名下运行

## 常见问题

### 如何在无痕模式(Incognito)下使用？
默认情况下，Chrome 扩展在无痕模式下是禁用的。如需开启：
1. 在浏览器地址栏输入 `chrome://extensions/`
2. 找到本插件，点击 **详细信息 (Details)** 按钮
3. 向下滚动，找到 **在无痕模式下启用 (Allow in Incognito)** 选项
4. 点击开关将其启用即可
