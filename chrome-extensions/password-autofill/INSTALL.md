# 安装指南

## 快速安装

### 方法一：开发者模式安装（推荐）

1. **打开扩展程序页面**
   - 在 Chrome 地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开关，启用开发者模式

3. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择插件所在的文件夹：`chrome-extensions/password-autofill`
   - 点击"选择文件夹"

4. **验证安装**
   - 在扩展程序列表中找到"Medtronic Smart Assistant"
   - 确认状态为"已启用"
   - 在 Chrome 工具栏中应该能看到插件图标

### 方法二：打包安装

1. **打包扩展程序**
   - 在 `chrome://extensions/` 页面
   - 点击"打包扩展程序"
   - 选择插件文件夹
   - 生成 `.crx` 文件

2. **安装打包文件**
   - 将 `.crx` 文件拖拽到 Chrome 窗口
   - 点击"添加扩展程序"

## 首次配置

### 1. 配置 Windchill
1. 点击插件图标打开弹出窗口
2. 点击 Windchill 卡片右侧的 ▼ 图标
3. 输入用户名和密码
4. 点击"💾 保存设置"

### 2. 配置 EHR
1. 点击 EHR 卡片右侧的 ▼ 图标
2. 输入用户名和密码
3. 点击"💾 保存设置"

### 3. 配置 Cornerstone（可选）
1. 点击 Cornerstone 卡片右侧的 ▼ 图标
2. 设置自动点击次数（默认30次）
3. 点击"💾 保存设置"
4. 右键点击插件图标，开启"自动点击"功能

### 4. 配置定时启动（可选）
1. 点击"定时启动"卡片的开关
2. 设置启动时间（默认 09:00）
3. 点击"⏰ 保存时间"

## 权限说明

插件需要以下权限：

- **storage**: 保存配置信息
- **tabs**: 打开和管理标签页
- **activeTab**: 访问当前标签页
- **contextMenus**: 创建右键菜单
- **webRequest**: 处理 HTTP 请求（用于 Basic Auth）
- **alarms**: 定时任务
- **notifications**: 桌面通知

所有权限仅用于插件功能，不会收集或上传任何个人信息。

## 在无痕模式下使用

1. 进入 `chrome://extensions/`
2. 找到"Medtronic Smart Assistant"
3. 点击"详细信息"
4. 向下滚动，找到"在无痕模式下启用"
5. 点击开关启用

## 卸载

如需卸载插件：

1. 进入 `chrome://extensions/`
2. 找到"Medtronic Smart Assistant"
3. 点击"移除"按钮
4. 确认卸载

注意：卸载后所有保存的配置信息将被清除。

## 更新

### 手动更新
1. 下载最新版本的插件文件
2. 在 `chrome://extensions/` 页面
3. 点击"Medtronic Smart Assistant"的"重新加载"按钮
4. 或者移除旧版本，重新安装新版本

### 自动更新（Chrome Web Store 版本）
- 插件会自动检查更新
- 有新版本时会自动下载并安装
- 重启浏览器后生效

## 常见问题

### Q: 安装后看不到插件图标？
A: 点击 Chrome 工具栏右侧的拼图图标，找到插件并点击"固定"。

### Q: 插件无法加载？
A: 
1. 确认已启用开发者模式
2. 检查文件夹路径是否正确
3. 查看是否有错误提示
4. 尝试重新加载插件

### Q: 配置信息丢失？
A: 
1. 检查插件是否被禁用
2. 确认没有清除浏览器数据
3. 重新配置并保存

### Q: 功能不工作？
A: 
1. 检查是否在支持的网站上使用
2. 查看浏览器控制台是否有错误
3. 尝试重新加载插件
4. 检查 Chrome 版本是否过旧
